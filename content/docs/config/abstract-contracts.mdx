title: Abstract Contracts
description: Common contract addresses and ABIs for tokens and DEX infrastructure on Abstract
---

## Installation

```bash
npx shadcn@latest add "https://agw-reusables.vercel.app/r/abstract-contracts.json"
```

## Usage

```tsx
import { ABSTRACT_CONTRACTS, ABSTRACT_NETWORKS } from "@/components/abstract-contracts"

// Access contracts directly
const wethAddress = ABSTRACT_CONTRACTS.tokens.weth.addresses.mainnet
// "0x3439153EB7AF838Ad19d56E1571FBD09333C2809"

const usdcTestnet = ABSTRACT_CONTRACTS.tokens.usdc.addresses.testnet  
// "0xe4C7fBB0a626ed208021ccabA6Be1566905E2dFc"
```
