## Installation

Ensure you have installed and setup the [AGW Provider](/docs/essentials/agw-provider) wrapper component first.

```bash
npx shadcn@latest add "https://agw-reusables.vercel.app/r/connect-wallet-button.json"
```

## Usage

```tsx
import { ConnectWalletButton } from "@/components/connect-wallet-button"

export default function ConnectWalletButtonDemo() {
  return (
    <ConnectWalletButton />
  )
}
```

### Custom Styling

```tsx
import { ConnectWalletButton } from "@/components/connect-wallet-button"

export default function CustomConnectWalletButton() {
  return (
    <ConnectWalletButton className="bg-blue-600 hover:bg-blue-700" />
  )
}
```

### Custom Dropdown Items

```tsx
import { ConnectWalletButton } from "@/components/connect-wallet-button"
import { DropdownMenuItem, DropdownMenuSeparator } from "@/components/ui/dropdown-menu"

export default function CustomConnectWalletButton() {
  return (
    <ConnectWalletButton 
      customDropdownItems={[
        <DropdownMenuSeparator key="separator" />,
        <DropdownMenuItem key="profile">
          View Profile
        </DropdownMenuItem>,
        <DropdownMenuItem key="settings">
          Settings
        </DropdownMenuItem>,
        <DropdownMenuItem key="disconnect" className="text-destructive">
          Disconnect
        </DropdownMenuItem>
      ]}
    />
  )
}
```

## What's included

The installation command adds the following file to your project:

- [`connect-wallet-button.tsx`](#connect-wallet-buttontsx) - The button component with all states and functionality

### `connect-wallet-button.tsx`

A button that prompts users to connect to the application with their [Abstract Global Wallet](https://docs.abs.xyz/abstract-global-wallet/overview).
