Built on top of the [shadcn/ui](https://ui.shadcn.com/) library, AGW Reusables uses the [shadcn CLI](https://ui.shadcn.com/docs/cli) to add snippets of code to your project and install any required dependencies to use them.

## Get Started

Follow the steps below to start using AGW Reusables.

### Setup your project

Create a new project or configure an existing one using the `shadcn init` command:

```bash
npx shadcn@latest init
```

### Install the AGW Provider

Install your first component, the [AGW Provider](/docs/essentials/agw-provider):

```bash
npx shadcn@latest add "https://agw-reusables.vercel.app/r/agw-provider.json"
```

### Add the wrapper component

Wrap your application in the installed component inside `app/layout.tsx`:

```tsx title="app/layout.tsx" showLineNumbers {1,6,10}
import { NextAbstractWalletProvider } from "@/components/agw-provider";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <NextAbstractWalletProvider>
        <body className={`${geistSans.variable} ${geistMono.variable} antialiased`}>
          {children}
        </body>
      </NextAbstractWalletProvider>
    </html>
  );
}
```
