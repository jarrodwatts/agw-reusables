## Introduction

AGW Reusables is a collection of components and utilities for building applications on [Abstract](https://abs.xyz).
It allows you to add features, components, and utilities to your application using single commands,
with only one requirement: **your app uses [Next.js](https://nextjs.org/docs)**.

Built on top of the [shadcn/ui](https://ui.shadcn.com/) library, AGW Reusables uses the [shadcn CLI](https://ui.shadcn.com/docs/cli) to add snippets of code to your project and install any of the required dependencies.

## Get Started

Follow the steps below to start using AGW Reusables.

### 1. Setup your project

Create a new project or configure an existing one using the `shadcn init` command:

```bash
npx shadcn@latest init
```

### 2. Install & Setup Provider

Install your first component, the [AGW Provider](/docs/essentials/agw-provider):

```bash
npx shadcn@latest add "https://agw-reusables.vercel.app/r/agw-provider.json"
```

Wrap your application in the `NextAbstractWalletProvider` component in `app/layout.tsx`:

```tsx showLineNumbers {1,6,10}
import { NextAbstractWalletProvider } from "@/components/agw-provider";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <NextAbstractWalletProvider>
        <body className={`${geistSans.variable} ${geistMono.variable} antialiased`}>
          {children}
        </body>
      </NextAbstractWalletProvider>
    </html>
  );
}
```
