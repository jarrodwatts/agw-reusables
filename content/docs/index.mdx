# Welcome to AGW Reusables

**A curated collection of reusable React components built with shadcn/ui.**

AGW Reusables provides high-quality, customizable components that you can copy and paste directly into your applications. Each component is built with accessibility, performance, and developer experience in mind.

## What's Different

Unlike traditional component libraries, AGW Reusables gives you:

- **Full ownership** - Copy the code, it's yours to modify
- **Zero dependencies** - No bloated packages or lock-in
- **Tailwind CSS** - Modern utility-first styling
- **TypeScript** - Full type safety out of the box
- **shadcn/ui compatible** - Works seamlessly with the shadcn ecosystem

## Quick Start

1. **Initialize shadcn/ui** in your project:
   ```bash
   npx shadcn@latest init
   ```

2. **Browse components** in the sidebar and find what you need

3. **Copy and paste** the component code into your project

4. **Customize** as needed for your use case

## Component Categories

- **UI Components** - Basic building blocks like buttons, inputs, cards
- **Layout Components** - Grid systems, containers, navigation
- **Interactive Components** - Modals, dropdowns, accordions
- **Data Display** - Tables, lists, charts
