# Complex Component

A comprehensive example showing how to build complex components using hooks, utility libraries, and multiple component composition. This component fetches and displays Pokémon data.

## Usage

```tsx
import PokemonPage from "@/components/complex-component/page"

export function PokemonDemo() {
  return (
    <div className="container mx-auto py-8">
      <PokemonPage />
    </div>
  )
}
```

## Features

- **Custom Hooks** - Uses `usePokemonImage` hook for image URL generation
- **Utility Functions** - Includes `getPokemonList` and `getPokemon` API utilities
- **Component Composition** - Combines `PokemonCard` and `PokemonImage` components
- **API Integration** - Fetches data from the PokéAPI
- **Zod Validation** - Type-safe API response validation
- **React Cache** - Uses React's cache for optimized data fetching

## Architecture

This component demonstrates a typical complex component structure:

### Components
- **PokemonCard** - Individual Pokémon card component
- **PokemonImage** - Handles Pokémon image display with fallbacks

### Hooks
- **usePokemonImage** - Custom hook for generating Pokémon image URLs

### Utilities
- **getPokemonList** - Fetches list of Pokémon from API
- **getPokemon** - Fetches individual Pokémon details

## API Integration

The component integrates with the [PokéAPI](https://pokeapi.co/) to demonstrate:

- **Type-safe API calls** using Zod schemas
- **Error handling** with try/catch blocks
- **Data caching** using React's cache function
- **Loading states** and fallbacks

## File Structure

```
complex-component/

├── components/

│   └── pokemon-image.tsx # Image component

```

This structure demonstrates how to organize complex components with multiple files and responsibilities.
