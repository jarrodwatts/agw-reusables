# Example Form

A contact form component that demonstrates form handling, validation with Zod, and integration with shadcn/ui form components.

<ComponentPreview name="example-form" />

## Installation

First, install the required dependencies:

```bash
npm install zod react-hook-form @hookform/resolvers
```

Then copy and paste the component code:

<ComponentSource name="example-form" />

## Usage

```tsx
import { ExampleForm } from "@/components/example-form"

export function ContactPage() {
  return (
    <div className="max-w-md mx-auto">
      <ExampleForm />
    </div>
  )
}
```

## Features

- **Zod Validation** - Type-safe form validation with Zod schema
- **React Hook Form** - Efficient form handling with minimal re-renders
- **shadcn/ui Components** - Uses Button, Input, Label, Textarea, and Card
- **Error Handling** - Displays validation errors inline
- **Responsive Design** - Mobile-friendly form layout

## Form Schema

The form uses a Zod schema for validation:

```typescript
const formSchema = z.object({
  name: z.string().min(2, "Name must be at least 2 characters"),
  email: z.string().email("Please enter a valid email address"),
  message: z.string().min(10, "Message must be at least 10 characters"),
})
```

## Customization

You can easily modify the form by:

- **Adding fields** - Extend the Zod schema and add new form inputs
- **Changing validation** - Update the schema rules as needed
- **Styling** - Modify Tailwind classes for different layouts
- **Submit handling** - Add your own form submission logic
