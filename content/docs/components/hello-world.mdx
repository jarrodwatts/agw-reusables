## Features

- **Simple Structure** - Demonstrates basic component patterns
- **Button Integration** - Uses the shadcn/ui button component
- **Clean Design** - Minimal styling with Tailwind CSS

This component serves as a starting point for understanding how components are structured in the AGW Reusables registry.
